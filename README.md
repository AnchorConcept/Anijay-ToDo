# Anijay To-Do App

A simple and responsive To-Do List web application built using **React**.

## 🔧 Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Tasks saved in localStorage
- Clean, mobile-friendly UI

## 🚀 Tech Stack

- React
- JavaScript
- HTML/CSS
